# VampireKnight
A bloodsucking will!

## What's this mod?

This mod, Vampire Knight, changes the gameplay of Hollow Knight by making the game more *difficult* or **easier** at times.

## What does it do?

If you're in a zone with enemies, your ***health*** will slowly decrease.. and to regain it you have to hit enemies (suck their BLOOD!).

## Else

This mod is kind of broken in some cases, like being able to hit Soul Master/Soul Tyrant when you normally can't. When I tried to fix this it would cause many more bugs that were not worth fixing for such a small occurence.

You can toggle this mod whenever you want and change the rate at which health decreases to whatever you wish!

This mod can be very fun in godhome and pantheons.
